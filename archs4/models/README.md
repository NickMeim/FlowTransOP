# ARCHS4 Models

This folder stores local ARCHS4 FlowTransOP model artifacts. These files are
not tracked because they are large and can exceed GitHub limits.

Typical contents generated by the ARCHS4 training scripts include:

- `fold_{fold}_normal.pt` and `fold_{fold}_permuted.pt`
- `fold_{fold}_normal_m2h.pt` and `fold_{fold}_permuted_m2h.pt`
- `full_ensemble_{id}_normal.pt`
- `fold_*_z_*.npy` latent arrays used for evaluation and diagnostics

Relevant generators in `learning/` include `train_ARCHS4_fold.py`,
`train_ARCHS4_fold_m2h.py`, and `train_ARCHS4_full_ensemble.py`.

Keep these files local, or distribute them through an external data archive or
Git LFS if pretrained weights need to be shared.
