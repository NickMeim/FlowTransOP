# MASH and Fibrosis Case-Study Scores

This folder stores the tabular outputs for the ARCHS4-trained FlowTransOP MASH
and fibrosis case studies.

Generated by:

- `learning/score_liver_mas_fibrosis_final_expression_mean.py`

The scoring workflow trains or applies raw-human PLSR models and compares mouse
liver perturbation profiles before and after FlowTransOP translation. Outputs
include predicted score differences, LOOCV diagnostics, and statistical summary
tables used by the case-study plots.

Figures are written to the nested `figures/` folder by
`postprocessing/plot_liver_mas_fibrosis_final_expression_mean.R`.
